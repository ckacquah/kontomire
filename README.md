# How to use

```console
$ git clone --recurse-submodules git@github.com:ckacquah/kontomire.git
$ cd kontomire
$ mkdir build
$ cd build
$ cmake ../
```

# Dependencies

- GLAD
- GLFW
- GoogleTest
- Imgui
- Implot
- GLM
