# How to use

```console
$ git clone --recurse-submodules git@github.com:ckacquah/Kontomire-Rendering-Engine.git
$ cd Kontomire-Rendering-Engine
$ mkdir build
$ cd build
$ cmake ../
```

# Dependencies

- GLAD
- GLFW
- GoogleTest
- Imgui
- Implot
