# How to use

```console
$ git clone --recurse-submodules git@github.com:ckacquah/OpenGL_Cpp_Template.git
$ cd OpenGL_Cpp_Template
$ mkdir build
$ cd build
$ cmake ../
```

# Dependencies

- GLAD
- GLFW
- GoogleTest
- Imgui
- Implot
- GLM
